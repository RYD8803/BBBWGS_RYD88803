# Overview
This project were aimed to analyzed three Streptococcus thermophilus strain (CF2_14, CF8_14, CF13_14) to find any SNP unique to each strain and to distinguish strain-specific SNPs in order to identify unique genetic characteristics of each strain.

# Workflow
## Data Retrieval
Retrieved from NCBI-SRA using SRA Toolkit prefetch (v3.4.1).

*S. thermophilus CF2_14* [https://www.ncbi.nlm.nih.gov/sra/SRR5310876] SRR5310876

*S. thermophilus CF8_14* [https://www.ncbi.nlm.nih.gov/sra/SRR11910258] SRR11910258

*S. thermophilus C13_14* [https://www.ncbi.nlm.nih.gov/sra/SRR11910376] SRR11910376

All SRA data were provided through BioProject [https://www.ncbi.nlm.nih.gov/bioproject/PRJNA336518]PRJNA336518 (Barnaba 2022).
